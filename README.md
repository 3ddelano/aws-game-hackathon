# aws-game-hackathon

Required:

- Node.js v22.5.1
- Bun: https://bun.sh/docs/installation


First copy `.env-sample` to `.env` update any values in the `.env` file if required.

To install dependencies:

```bash
bun install
```

To run server:

```bash
cd apps/server
bun dev
```

To run client:

```bash
cd apps/client
bun dev
```
